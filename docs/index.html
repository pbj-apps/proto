RpmService

RpmService

The RpmService provides access to resources related to Remote Patient Monitoring (RPM).

URLs

Scratch: heyrenee-v1-8jiz6fu6.uc.gateway.dev

Method NameRequest TypeResponse TypeDescription
ListRpmSchedules ListRpmSchedulesRequest ListRpmSchedulesResponse

ListRpmSchedules retrieves a list of a Patient's RpmSchedules. These schedules specify when a patient should use RPM devices to take different RPM measurements.
ListRpmMeasurements ListRpmMeasurementsRequest ListRpmMeasurementsResponse

ListRpmMeasurements retrieves a list of RpmMeasurements for a specific RpmSchedule. These measurements represent a single event where an RPM device was used to take a measurement. The RpmMeasurement contains a list of RpmResults where each individual result of the measurement event is listed.
